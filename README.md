# 🚀 My Portfolio

This is my personal portfolio website built using **Next.js** and **Tailwind CSS**. It showcases my skills, projects, and contact information.

## 🔥 Features
- Hero section with an introduction
- About me section
- Skills showcase
- Projects gallery
- Contact form
- Responsive design

## 🛠️ Technologies Used
- **React (Next.js)**
- **Tailwind CSS**
- **TypeScript**
